# Évolution du PIB français jusqu'au T1 2022

Ce jeu de données contient une série `ts` de l'évolution trimestrielle
du produit intérieur brut français. Toutes les infos complémentaires sur
cette série se trouve sur la page de la
[publication](https://www.insee.fr/fr/statistiques/2830547) sur le site
de l'[Insee](https://www.insee.fr/fr/accueil).

## Usage

``` r
ev_pib
```

## Format

Un ts unidimensionnel :

- start:

  le ts commence au T1 1970 mais la série de PIB ne commence qu'au T2
  1980.

- end:

  le ts finit au T3 2022 mais la série de PIB finit au T1 2022.

- frequency_ts:

  la fréquence est trimestrielle
